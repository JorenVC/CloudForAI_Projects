Stap 1)

Ga in command line naar de map waar de dockerfile staat.

Stap 2)

Run het commando: docker build . -t cloudai_project2_jvc

Stap 3)

Ga naar docker en run 'cloudai_project2_jvc'

Stap 4)

Ga naar docker container en run de container opnieuw en in de Logs zie je 
welke lijnen er in 'Processing' zijn. Als je in de files 'workdir_cloud_ai_project2'
download dan zie je dat deze persoon in 'Prediction_records.csv' zijn opgeslagen.
